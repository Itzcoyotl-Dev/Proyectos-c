# Proyectos en C

![Lenguaje](https://img.shields.io/badge/Lenguaje-C-blue)
![Estado](https://img.shields.io/badge/Estado-En%20progreso-yellow)
![Autor](https://img.shields.io/badge/Autor-Alvaro%20Ceballos-green)

Un conjunto de proyectos que refuerzan el manejo de estructuras
de decisión y ciclos, tipos de datos, strings, manejo de arreglos
tanto unidimensionales como bidimensionales y funciones por paso de valor.

## Objetivo
Proyectos desarrollados como parte de mi aprendizaje del lenguaje C,
enfocados en consolidar fundamentos de programación estructurada.

## Requisitos
- Compilador GCC
- Terminal / Consola

## Compilación
gcc nombre_archivo.c -o programa
./programa

## Proyectos

| Proyecto | Descripción | Dificultad |
|----------|-------------|------------|
| Par_Impar        | Muestra si el número dado por el usuario es par o impar | ⭐ |
| Mayor_Menor | Busca el mayor y menor en un conjunto de números | ⭐ |
| Promedio | Calcula el promedio de un conjunto de calificaciones | ⭐⭐ |
| Calculadora | Menú interactivo con operaciones aritméticas | ⭐⭐ |
| Conversor | Menú interactivo para conversiones | ⭐⭐ |
| Registro_de_Nombres | Arreglo bidimensional de strings de nombres | ⭐⭐ |
| Validador_de_Acceso | Manejo de credenciales con strings de una matriz | ⭐⭐⭐ |
| Filtro_de_Acceso | Manejo de acceso con arreglos de ID | ⭐⭐⭐ |
| Juego_del_Gato (Tic-Tac-Toe) | Manejo complejo de arreglos, funciones y strings | ⭐⭐⭐ |
